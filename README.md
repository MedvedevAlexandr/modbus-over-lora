# Modbus over LoRa

**Беспроводная передача данных по протоколу Modbus RTU с использованием LoRa**  
Проект демонстрирует, как организовать надёжный обмен данными между двумя устройствами на базе Arduino через радиоканалы LoRa, инкапсулируя Modbus-кадры. Подходит для систем удалённого мониторинга, промышленной автоматизации и IoT-приложений с низким энергопотреблением.

---

## 📡 Особенности

- Реализация **Modbus RTU** поверх радиосвязи LoRa.
- Работа в режиме **Master** (запросы) и **Slave** (ответы) на двух независимых Arduino.
- Использование **CRC-16 Modbus** для контроля целостности данных.
- Настраиваемые параметры LoRa: частота, **Spreading Factor (SF)**, полоса пропускания (BW).
- Код полностью совместим с Arduino Nano/Uno и модулями LoRa на чипе **SX1276/SX1278** (например, Ra-02).
- Поддержка функции **0x03** (чтение holding registers) – легко расширяется под другие функции.

---

## 🧰 Аппаратное обеспечение

- 2× **Arduino Nano** (или Uno, с учётом напряжений)
- 2× **LoRa-модуль Ra-02** (433 МГц или 868 МГц)
- Макетные провода «мама-мама»
- (Опционально) стабилизатор 3.3 В (если модуль не питается от 3.3V Arduino)

### Схема подключения (на одно устройство)

| LoRa Ra-02 | Arduino Nano |
|------------|---------------|
| SCK        | D13           |
| MISO       | D12           |
| MOSI       | D11           |
| NSS        | D10           |
| RST        | D9            |
| DIO0       | D2            |
| VCC        | 3.3V          |
| GND        | GND           |
